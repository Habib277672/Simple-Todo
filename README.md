📋 Simple Todo (React)

A clean and easy‑to‑use Todo application built with React.js using local component state.

👉 Live Preview: https://habib277672.github.io/Simple-Todo/

🚀 Overview

This project is a simple Todo list application developed with React. It allows users to:

📝 Add Todo items

❌ Delete Todo items

✔️ Mark items as completed

🧹 Maintain a basic list using React state

It’s a great demonstration of React fundamentals including useState, component‑based design, and event handling.

🧠 Key Features

Add Tasks — Enter a task and add it to the list

Delete Tasks — Remove any task

Complete Tasks — Toggle tasks between completed/incomplete

Simple UI — Clean interface with minimal styling

🛠️ Built With

React.js — Frontend library

HTML — Structure

CSS — Styling

JavaScript — App logic

GitHub Pages — Deployment
